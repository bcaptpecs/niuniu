# 🃏 广东纸牌牛牛 · 网页版

一个纯HTML/CSS/JavaScript实现的广东纸牌牛牛游戏，具有发牌动画和完全响应式设计。

## 🎮 游戏特色

- **真实广东牛牛规则**：A=1点，J/Q/K=10点
- **发牌动画**：一张一张交替发牌，每张牌间隔2秒
- **响应式设计**：手机无需滚动即可完整查看游戏
- **完整牌型判断**：五花牛 > 炸弹 > 五小牛 > 牛牛 > 有牛 > 无牛

## 📱 技术特点

- **纯前端实现**：无需后端服务器
- **CSS动画**：牌从右侧飞入动画
- **移动端优化**：适配320px以上所有屏幕
- **自动化部署**：GitHub + Vercel 自动部署

## 📄 游戏规则

1. 点击"🀄️ 发牌"按钮开始游戏
2. 系统自动发5张牌给玩家和庄家
3. 从5张牌中选3张凑成10的倍数（10/20/30）
4. 剩余2张牌点数之和取个位数即为牛数
5. 牛数为0时为"牛牛"，为1-9时为"牛几"

## 🎯 特殊牌型

| 牌型 | 说明 | 大小顺序 |
|------|------|----------|
| 五花牛 | 五张牌都是J/Q/K | 最大 |
| 炸弹 | 四张相同点数的牌 | 第二 |
| 五小牛 | 五张牌点数≤5且总点数≤10 | 第三 |
| 牛牛 | 三张凑10倍数，剩余两张和个位为0 | 第四 |
| 有牛 | 有牛数（牛1-牛9） | 第五 |
| 无牛 | 无法组成10倍数 | 最小 |
